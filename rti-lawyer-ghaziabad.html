Common RTI Mistakes

?????? ????: ?? ??????? ?? ????

Broad / Vague Queries

Asking "provide all documents of the department" often gets rejected. Always ask specific, itemized questions.

Asking "Why"

Under RTI, you extract "information" (facts/files), not opinions. Don't ask questions starting with "Why" or "How come".

RTI Act FAQs

?????? ????? ???? ???? ???? ?????? - ????? ??????
Who can file an RTI application?
Any citizen of India can file an RTI application. Corporations or NGOs cannot file RTIs directly as "citizens," but their representatives can file as individuals. Advocate Gunjan specializes in framing these requests for maximum impact.
What is the time limit for getting information?
Standard info must be provided within 30 days. If the info concerns "Life or Liberty," it must be provided within 48 hours. If a Third Party is involved, the limit is 40 days.
What is a "Deemed Refusal"?
If the PIO fails to reply within 30 days, it is legally considered a "Deemed Refusal." This allows the applicant to file a First Appeal immediately without waiting any longer.
Can RTI be rejected?
Yes, information exempt under Section 8 (National Security, Fiduciary relation, Privacy etc.) can be rejected. However, many PIOs use these as excuses. We help challenge arbitrary rejections before the Commission.
Do I need to state a reason for seeking info?
No. Under Section 6(2), an applicant is NOT required to give any reason for requesting information or any other personal details except those necessary for contacting them.
